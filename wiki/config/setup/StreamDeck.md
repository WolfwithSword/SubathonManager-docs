# StreamDeck Extension

A StreamDeck extension is available which lets you control SubathonManager directly from your Elgato Stream Deck. All commands available via chat or websocket can be done via streamdeck - adding wheel spins, pausing/resuming, adding time, etc.

---

## Setup

To setup or update the streamdeck plugin, in SubathonManager go to **Settings** > **External Software** > **StreamDeck** and click "Download Stream Deck Plugin".

Download the file linked and double click it to install into your StreamDeck software as a plugin.

---

## StreamDeck Configuration

Once the plugin is installed, you can add buttons for each command right in the UI. For only one do you need to setup the port of SubathonManager.
