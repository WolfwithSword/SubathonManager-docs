# Pally.GG

To integrate Pally.GG, you can get your personal [API key here](https://pally.gg/dashboard/settings/api-keys) to enter into SubathonManager, under **External Services**.

Then you can select a single "room". The room is the name of whichever page you want to track donations for, should you have more than one. If you leave it blank, it will track all of the pages you manage.
