# Subathon Manager

<div class="home-hero" markdown>
![SubathonManager icon](assets/icon.png)
</div>

**SubathonManager** is a feature-rich tool for streamers with a subathon timer, goals + event manager, and overlay host - integrating Twitch, YouTube, KoFi, GoAffPro, and many more to.

All completely local, opensource, and made with no AI.
